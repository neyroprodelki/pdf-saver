# 📑 PDF-saver: Автоматизация для юристов

Скрипт на Python для массового сохранения PDF-документов, открытых в браузере. Идеально подходит для работы с порталами **Кад.Арбитр**, **ГИС ЖКХ** и другими государственными сервисами, где нужно выкачивать сотни файлов за раз.

## 🚀 Зачем это нужно?
Когда у вас открыто 200+ вкладок с определениями судов, сохранять каждый файл вручную (Ctrl+S -> Enter -> Ctrl+W) — это часы рутины. Этот скрипт имитирует действия пользователя и делает всю работу за вас.

## ✨ Основные функции
- **Массовое сохранение:** Проходит по всем открытым вкладкам.
- **Метод "Конвейер":** Сохраняет файл и сразу закрывает вкладку, освобождая оперативную память.
- **Fail-Safe:** Мгновенная остановка скрипта при перемещении мыши в угол экрана.
- **Интерфейс:** Ввод количества вкладок через простое диалоговое окно.

## 📦 Установка и запуск

### Вариант 1: Исполняемый файл (.exe)
1. Скачайте архив из раздела [Releases](ссылка_на_релиз).
2. Запустите `pdf_marathon.exe`.

### Вариант 2: Из исходного кода (Python)
1. Установите необходимые библиотеки:
   ```bash
   pip install pyautogui
   ```
2. Запустите скрипт:
   ```bash
   python main.py
   ```

## 🛠 Инструкция
1. Откройте браузер. С помощью средней кнопки мыши (колесика) откройте нужные документы в новых вкладках.
2. Запустите программу и введите количество документов.
3. У вас будет **5 секунд**, чтобы переключиться на окно браузера и кликнуть по первой вкладке.
4. **Важно:** Не трогайте мышь и клавиатуру до окончания процесса.

> 🆘 **Как остановить?** Если что-то пошло не так, резко переместите курсор мыши в **верхний левый угол** экрана.
